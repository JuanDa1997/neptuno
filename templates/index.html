¿Qué es Neptuno?


Neptuno es una herramienta desarrollada para ayudar

que se encargará de gestionar el control parcial de tu computadora.

Debes contar con una conexión a internet estable para poder usar

todas las funcionalidades de la herramienta, una cámara web y un micrófono.
